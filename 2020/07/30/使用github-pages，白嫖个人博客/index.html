使用github pages，白嫖个人博客

发表于2020-07-30|更新于2022-03-15
|阅读量:

总结自网上的教程，使用hexo + github pages，快速搭建个人博客（windows操作系统）
使用github pages搭建个人博客，可以免去购买服务器的钱。

hexo

1.node.js

在node.js官网直接下载node.js安装包，无脑下一步安装。

2.cnpm工具(使用淘宝的镜像源)

3.hexo blog框架

使用hexo搭建blog

1.新建一个文件夹，并git bash该文件夹

2.初始化文件夹

然后会得到如下的文件夹目录：

此时已经搭建好了博客，默认发布了一篇名为hello world的文章

github pages

1.在blog目录下安装git部署插件

cnpm install --save hexo-deployer-git

2.创建github仓库

在github新建一个repository，命名为username.github.io

3.配置github仓库到本地

复制新建的repository的地址，然后打开blog文件夹的根目录下的_config.yml文件，修改里面deploy的内容（在最后）。repo对应就是repository的地址。

4.部署到github仓库
